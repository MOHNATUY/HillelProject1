# ✅ Список Завдань для NextDrop

Мікросервісний ігровий магазин потребує реалізації кількох ключових компонентів. Список задач поділено на блоки.

---

## 🎯 Основні завдання

- Створити репозиторій `nextdrop`
- Додати захист гілки `main`
- Налаштувати GitHub Actions (lint, tests)
- Реалізувати `users-service` з FastAPI + JWT
- Реалізувати `games-service` з PostgreSQL
- Додати `orders-service` для обробки покупок
- Завантаження зображень (обкладинки) у S3
- Повідомлення користувачам через RabbitMQ
- MkDocs документація (для кожного сервісу)

---

## 🔐 Безпека

- JWT авторизація для users-service
- Захист доступу до приватних маршрутів
- Валідація ролей (admin / user)

---

## 💻 CI/CD

- Працюючий `ci.yml` в `.github/workflows`
- PR має запускати тести та лінт
- Мінімум 2 апруви перед merge
- Docker-образи для кожного сервісу

---

## 🧪 Тестування

- Unit-тести для users-service
- Інтеграційні тести між services
- Навантажувальне тестування (опціонально)
